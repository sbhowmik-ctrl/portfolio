## MY portfolio

Next.js application

### Structure

- `src/app/layout.tsx`: App layout and metadata
- `src/app/page.tsx`: Landing page
- `src/app/globals.css`: Global styles (Tailwind v4)

### Requirements

- Node 18+
- pnpm (recommended) or npm/yarn

### Getting Started

```bash
pnpm install
pnpm dev
```
